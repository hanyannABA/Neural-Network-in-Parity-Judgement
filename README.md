# Neural Network in Parity Judgement
# 判断奇偶数的神经网络模型

*It's a marvellous idea for exploring the possibility of judging whether a number is odd or even using a simple neural network. Please enjoy it!*
*这是一个富有创造性的想法！我们可以训练一个简单的神经网络去判断一个数的奇偶性。这个看似“脱裤子放屁”的行为实则带来了诸多乐趣！*

---

## Files Structure
## 文件目录

- **DataSet/**  
  - `train.csv`：训练数据  
  - `validation.csv`：验证数据  
  - `test.csv`：测试数据  
  - `DataSet_generator.ipynb`：数据集生成脚本  

- **model/**  
  - `parity_model.keras`：训练好的模型文件（使用方法参考 `test.ipynb`）  

- **Net.ipynb**  
  神经网络的主要训练与测试代码
